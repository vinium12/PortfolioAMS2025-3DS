# PortfolioAMS2025-3DS

Portfolio 2025 do 3º ano do curso de Desenvolvimento de Sistemas (AMS) da Etec, desenvolvido em parceria com o projeto P-TECH. O documento reúne todas as atividades, registros, documentações e horas estabelecidas pela empresa IBM, com o objetivo de comprovar e evidenciar o desenvolvimento e a conclusão das competências adquiridas ao longo do programa.
